# SILedger
SILedger adalah solusi akuntansi modern berbasis R yang dirancang untuk mengotomatisasi pemindahan data dari Jurnal Umum ke Buku Besar. Melalui sistem pencatatan real-time, setiap transaksi yang masuk akan langsung memperbarui seluruh laporan keuangan secara otomatis, memastikan pengelolaan finansial yang sistematis, akurat, dan efisien.
# 🎯 Tujuan
* Menyediakan software akuntansi gratis yang dapat digunakan untuk pencatatan keuangan bagi UMKM.
* Mengembangkan program berbasis R yang mampu melakukan pencatatan dan pengelolaan General Ledger (Buku Besar) secara cepat dan mudah digunakan.
* Menyediakan software akuntansi yang dapat digunakan mahasiswa belajar untuk melakukan pencatatan keuangan secara digital dan sistematis.
# ✨ Fitur Utama
* Journal Voucher
* List of Journal Voucher
* Account History
* Subsidiary Ledger
* List of Account Budget
* Account Balance
